About This Project

The Plantation Monitoring System is a web-based platform developed as a Final Year Project at Crescendo International College. It addresses key challenges commonly faced by oil palm plantations‚Äîsuch as the lack of real-time visibility , inconsistent worker attendance tracking, and limited tools for analyzing crop performance across multiple estates.

This centralized system enables plantation administrators to monitor daily operations, track worker attendance , visualize crop yield trends, and manage field data all in one place. Designed specifically for plantation owners, managers, and supervisors, the system enhances operational transparency , improves productivity, and supports data-driven decision-making.

Core Features

Worker Monitoring

Admins can track which workers are active daily based on attendance records and performance charts.

Attendance Tracking

Keep the Attendance record in the website. Time data is stored for admin review.

Crop Monitoring

Admins can track yield records, seasonal changes, and crop health over time.

Map Integration

Supports GeoTIFF/COG image overlays and KML to visualize plantation zones on an interactive map.

Role-based Access Control

Only authorized users such as Admin and Manager can access and operate the system. This ensures data privacy and operational control.

Centralized Reporting

All crop records, attendance logs, and performance metrics are stored and presented in one place for easier analysis and historical review.
